# Rock Paper Scissors

My first JavaScript project, built as part of 
The Odin Project Foundations curriculum.

## About

A console-based Rock Paper Scissors game played 
in the browser's developer console. The player 
is prompted for their choice each round, and the 
computer picks randomly. Best of 5 rounds.

## What I built

- Random computer choice generation
- Player input via browser prompt
- Round-by-round win/lose/draw logic
- Score tracking across 5 rounds
- Final winner declaration

## What I learned

- Writing and calling functions in JavaScript
- Using switch statements for multiple conditions
- Nested functions and variable scope
- Debugging with console.log and Chrome DevTools
- Thinking through logic before writing code
